## Install

Prerequisites:
 * PHP >= 5.5.9 or PHP 7
 * MySQL >= 5.5
 * [Composer](https://getcomposer.org/doc/00-intro.md#installation-linux-unix-osx)


1. Clone repository
2. Run composer install


 ```bash
 $ git clone -b 4-develop https://github.com/bedita/bedita.git
 $ cd bedita
 $ composer install
 ```
